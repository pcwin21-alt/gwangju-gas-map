📖 사용 안내
이 지도는?
광주광역시에서 광주상생카드 또는 온누리상품권으로 결제 가능한 주유소를 한눈에 볼 수 있는 지도입니다.
상생카드만 온누리만 둘 다 가능
주요 기능
마커 클릭 → 주유소 이름·주소, 결제 가능 수단, 현재 유가(휘발유·경유·LPG) 확인
검색창에 주유소명이나 주소를 입력하면 해당 위치로 이동합니다.
상생카드 / 온누리 / 전체 버튼으로 원하는 결제수단의 주유소만 볼 수 있습니다.
구 필터(광산구·남구·동구·북구·서구)로 원하는 지역만 볼 수 있습니다.
👑 저렴한 순위 → 현재 보이는 주유소 중 휘발유·경유·LPG 가격 TOP 5를 확인합니다.
📍 내 위치 → 현재 위치를 지도 중심으로 이동합니다. (브라우저 위치 권한 필요)
↺ 초기화 → 필터·검색·지도 위치를 한 번에 원래대로 되돌립니다.
누락·오류 제보
지도에 없는 주유소, 잘못된 정보가 있다면 + 제보 버튼을 눌러 알려주세요. 지도에서 위치를 직접 찍거나 주소를 입력하고, 사진도 첨부할 수 있습니다.
이메일로도 신고 가능 → very-speking@naver.com
데이터 출처
상생카드 주유소: 광주시 공공데이터 / 온누리상품권 가맹점: 공공데이터포털 / 유가 정보: 오피넷(한국석유공사)
데이터는 주기적으로 업데이트되며, 실제와 차이가 있을 수 있습니다.
